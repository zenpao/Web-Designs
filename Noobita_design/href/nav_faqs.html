HTML / CSS Corner

How to run my HTML file?

Ans. Drag your HTML file to your web browser or simply double click it.

How do I solve my HTML file not displaying properly?

Ans. Check whether you missed closing a <tag></tag> or you might've not placed your contents inside the <body></body> element.

How do I solve my HTML file not displaying CSS properly?

Ans. Check whether you missed closing a <tag></tag> or you might've not placed your imported your CSS inside the <head></head> element. Also check whether the CSS property you included is existent and compatible with an element.

How to import my external CSS inside my HTML?

Ans. Place and modify this line of code inside the <head></head> element:

<rel="stylesheet" type="text/css" href="YourcssFile.css" media="all">

Bootstrap Corner

How do I solve my HTML file not displaying Bootstrap properly?

Ans. Check whether you missed closing a <tag></tag> or you might've not imported your Bootstrap CSS inside the <head></head> element. Also check whether you included the jQuery correctly first and Bootstrap JS file inside the <body></body> element.

How to import my Bootstrap CSS inside my HTML?

Ans. Place and modify this line of code inside the <head></head> element:

<rel="stylesheet" type="text/css" href="bootstrap-3.3.7-dist/css/bootstrap.css" media="all">

How to import my Bootstrap JS inside my HTML?

Ans. Place and modify this line of code inside the <body></body> element:

<script type="text/javascript" src="bootstrap-3.3.7-dist/js/bootstrap.js"></script>

jQuery / JavaScript Corner

How do I solve my HTML file not displaying jQuery properly?

Ans. Check whether you might've not imported your jQuery inside the <body></body> element. Also check whether you included the jQuery correctly first and other JS file inside the <body></body> element.

How do I solve my HTML file not displaying JavaScript properly?

Ans. Check whether you might've not imported your JavaScript inside the <body></body> element. Also check whether you included the jQuery correctly first and other JS file inside the <body></body> element.

How to import my jQuery inside my HTML?

Ans. Place and modify this line of code inside the <body></body> element:

<script type="text/javascript" src="jquery-2.1.4.js"></script>

How to import my JavaScript inside my HTML?

Ans. Place and modify this line of code inside the <body></body> element and after a jQuery:

<script type="text/javascript" src="YourjsFile.js"></script>

Side notes

Do I need internet in developing a webpage?

Ans. Yes and no. You don't need internet in creating a prototype however you do need to be online when you are seeking updated documentation, solution and finally deploying your project.

Why is the HTML skeleton longer than what I find in the internet?

Ans. The HTML skeleton provided here is as close as it can get as a complete file but it is safe to omit or disregard the things you don't have use for.

Why do you mostly use Bootstrap 3 instead of Bootstrap 4?

Ans. For the reason that I am more accustomed to Bootstrap 3 and sure Bootstrap 4 has a lot of features and improvements but it isn't bad either to use Bootstrap 3 as it is still being maintained by the Bootstrap team. It all goes down to your own preference and to what is required of you in building the project.

How about Databases?

Ans. I had plans on including a guide for implementing data-storing capabilities for a webpage however where this webpage is hosted does not support the database that I am planning to use which is NoSQL / MongDB.

Ans. Because this guide is not a full pledge course for web development. There are sections which are lacking, outdated or not found. How about those?

Why is it named noobita?

Ans. As mentioned in the Definition of Terms, its name was inspired by combining the slang for "newbie" used by gamers as "noob" and the name of the male human protagonist "Nobita" in the hit 70s Japanese TV show "Doraemon".
