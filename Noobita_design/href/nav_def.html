Definition of Terms

Term Definition
Bluish color Refers that the subject included has a moderate learning curve. It may also refer as an "informative" property for design purposes.
Bootstrap Refers to the most popular CSS and JS library for building responsive webpages.
Brownish color Refers that the subject included has are mere information and executions for a platform. It may also refer as a "warning" property for design purposes.
Cheatsheet Refers to a list that acts as a reference.
Code editor Refers to a more advance note editor that is catered to suit a programmer's needs in writing code.
CSS "Cascading Style Sheet" are files used to design HTML content.
Documentation Refers to a transposed explanation for the usage of a particular subject. Typically explains the usage of a machine or software program.
Git Refers to a distributed version-control system for tracking changes in your code during software development.
Globe Refers that the webpage involves redirecting online.
Glyphicons Refers to icons or specialized fonts useable through Bootstrap.
Greenish color Refers that the subject included is beginner-friendly. It may also refer as a "successful" property for design purposes.
HTML "Hypertext Markup Language" serves as an executable file to be displayed in a web browser.
HTML Skeleton Refers to the program code used routinely for developing web pages.
Import Refers to including code or files into your project.
Information Refers that the webpage only contains information.
JS "JavaScript" is a scripting programming language most used to add interactive features to webpages.
jQuery Refers to a JavaScript library with ready-made functions which helps simplify and minimize many common tasks done with JavaScript.
Library Refers to a collection of prebuilt codes.
Localization Refers to resources or assets stored in the user's machine or computer.
Migration Refers to changing whether updating or downgrading into another platform.
Mock-up Refers to the second process of Web Design that is a photorealistic version of the wireframe usually done using third party graphic softwares.
NodeJS Refers to an environment that executes JavaScript code outside of a web browser.
noobita Refers to the name of this webpage, inspired by combining the slang for "newbie" used by gamers as "noob" and the name of the male human protagonist "Nobita" in the hit 70s Japanese TV show "Doraemon". The noobita project was made as a quick reference for learning basic web design and for personal use.
Offline Refers to no internet connectivity.
Paperclip Refers that the webpage contains attachments or downloadables.
Project Refers to what you are coding for.
Properties Refers to the status of a particular code.
Prototype Refers to the third process of Web Design that is an improved version of the mock-up usually incorporating animations for presentation.
Reddish color Refers that the subject included has a complex learning curve. It may also refer as a "danger" property for design purposes.
Selectors Refers to selecting a specific line or block of code.
Source code Refers to the program code used.
Template Refers to a premade file.
Web browser Refers to computer program used to navigate the web and view web pages.
Web page Refers to a document intended to be viewed with a web browser.
Wireframe Refers to the first process of Web Design that is a low-fidelity sketch illustration a design usually done with a pen and paper.
zenpao Refers to the alias of the author of the webpage.
